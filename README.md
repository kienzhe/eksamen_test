# 📙 About <a name = "about"></a>
Vue Do is a sample to do list using Vue.js and Vuex for state management.

# 🏁 Get started <a name = "install"></a>
```
git clone https://github.com/gaserashraf/Vue-Do.git
```
```
cd Vue-Do
```
```
npm install
```
```
npm run serve
```

# 💻 Built using <a name = "tech"></a>
- **Vue js**
- **Vuex**
